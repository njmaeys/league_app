# simple_web_app_LoL
dependancy  
  - lpthw.web  
  - `pip install lpthw.web`  
  
# API Key
Get your api key from riot and put it in a file called `api_key.txt` in the templates/ dir  

# Run the app
cd to the LoL_simple_app dir then `python bin/app.py`  
currently the app runs on *localhost:8080/*
